Proyecto desarrollado en PHP y MySQL para la gestión de alumnos y notas.

## Tecnologías utilizadas

- PHP
- MySQL
- XAMPP
- HTML

## Instalación

1. Importar el archivo sistema_notas.sql en phpMyAdmin.
2. Copiar la carpeta del proyecto en htdocs.
3. Iniciar Apache y MySQL.
4. Abrir:

http://localhost/sistema_notas

## Funcionalidades

- Inicio de sesión
- Registro de alumnos
- Gestión de notas
- Cierre de sesión
